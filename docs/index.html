✷✶✦ Sirius ✦✶✷

Sirius is an embedded DSL for expressing algorithms with strong safety guarantees, while being easy to adopt and maintain.

These docs and the language itself are under development. Anything written in this style is aspirational and not fully implemented. Sirius in a nutshell:

  • Syntax comparable to simplified Rust
  • Immutability
  • Static type checking
  • Type inference within function boundaries
  • Ergonomic static array access checking
  • Array-oriented facilities like broadcasting
  • Functional facilities like lambdas, map, and fold
  • Easy to extend

You can try out Sirius with the in-browser demo or by cloning the git repository.
