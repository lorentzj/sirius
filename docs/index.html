✷✶✦ Sirius ✦✶✷

Sirius is a prototype array-oriented GPU programming language with strong type safety features.

These docs and the language itself are under construction. Anything written in this style is aspirational and not fully implemented. Sirius in a nutshell:

  • "Python with types"-style syntax
  • Static type checking and inference within function boundaries
  • Imperative with opt-in mutability and some functional features
  • Parametric polymorphism
  • Static array shape and access checks
  • Array-oriented facilities like broadcasting and fancy indexing
  • A fast GPU backend
  • Python bindings and Jupyter integration

You can try out Sirius with the in-browser demo or by cloning the git repository.

Learn more with a gradual intro to the type system design, the standard library docs, or the project roadmap.
